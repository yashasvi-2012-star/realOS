# ZENOS 
ZENOS is my first OS that is based on my idol- jennieeee it is a OS with 
draggable window u can custumize on your own !!!!

this project include :

🕒 Movable Live Clock — always running, and you can drag it anywhere
🎵 Spotify Playlist Panel — chill music right inside your workspace and it plays actual music 
📝 Notepad — quick notes whenever ideas hit it is also draggable 
🎨 Drawing Tool — sketch, write, or just doodle freely with changable brush size and custum colour to select 
🧮 Calculator — simple, fast, always there when you need it. also draggable 

Everything is designed to feel lightweight, interactive, and easy to use without overcomplicating things and you can even place them wherver you like .

ZENOS is basically a small creative workspace where you can work, think, and chill — all in one place but in my style!!!!
